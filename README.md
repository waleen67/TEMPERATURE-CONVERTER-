# 🌡️ Temperature Converter

A simple Python program to convert temperatures between Celsius and Fahrenheit with input validation and absolute zero checks.

## 🚀 Features

- Convert **Celsius → Fahrenheit**
- Convert **Fahrenheit → Celsius**
- **Input validation** (accepts c/C, f/F, celsius, fahrenheit)
- **Error handling** for non-numeric inputs
- **Absolute zero validation**:
  - ❄️ Celsius: < -273.15°C
  - ❄️ Fahrenheit: < -459.67°F
- **Continuous conversion** option (y/n)

## 📋 Requirements

- Python 3.x
- No external libraries required
